

## Empezando

Primero, ejecute el servidor de desarrollo:

en bash
npm ejecutar dev
en Mac
desarrollador de hilo

después de instalar nextjs en su máquina, abra [http://localhost:3000](http://localhost:3000) con su navegador para ver el resultado.

Puede comenzar a editar la página modificando `pages/index.js`. La página se actualiza automáticamente a medida que edita el archivo.
